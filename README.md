# obaDesktop

obaDesktopはウェブ上で動作するデスクトップシステムです。
バックエンドAPIと通信して、ファイルマネージャーやサーバー管理ができることを目的としています。

![image](https://user-images.githubusercontent.com/29545778/218300165-7e65eb9c-52c3-4b15-bb6c-3c0212013f2e.png)


## セットアップ

必要なパッケージ群をインスコ

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## 起動する

```bash
# yarn
yarn 
```
