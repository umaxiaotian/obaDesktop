# obaDesktop

obaDesktopはウェブ上で動作するデスクトップシステムです。
バックエンドAPIと通信して、ファイルマネージャーやサーバー管理ができることを目的としています。

![image](https://user-images.githubusercontent.com/29545778/216829508-b8967849-dc26-4d46-82a9-08c423268447.png)


## セットアップ

必要なパッケージ群をインスコ

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## 起動する

```bash
# yarn
yarn 
```
